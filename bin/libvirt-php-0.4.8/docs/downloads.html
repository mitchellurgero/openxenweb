Official Releases

You can download the third official release, release 0.4.7, from: 

          http://libvirt.org/sources/php/libvirt-php-0.4.7.tar.gz

However if you prefer to download any of 3 latest versions of the previous official releases please click on the release you want to download:

http://libvirt.org/sources/php/libvirt-php-0.4.6.tar.gz
http://libvirt.org/sources/php/libvirt-php-0.4.5.tar.gz
http://libvirt.org/sources/php/libvirt-php-0.4.4.tar.gz

GIT source repository

You can clone the development version of libvirt-php from GIT repository: 

git clone git://libvirt.org/libvirt-php

If you want to contribute to the libvirt-php project please refer to Contributions.
