Roadmap for libvirt-php

This is the intended roadmap of libvirt-php project for the worth implementing features and/or bug fixes:

  • Write a Windows port to enable management from Windows-based servers
  • Implement traversal lookup for connection objects (e.g. to limit number of connection from PHP and free objects)
  • Write more examples to the website
  • Add support for managing snapshots and new libvirt features
